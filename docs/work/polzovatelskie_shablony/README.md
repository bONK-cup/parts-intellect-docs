В данной главе описано добавление пользовательских печатных шаблонов в **Мастер печати**.

::: info Примечание

В **Мастере печати** отображаются печатные шаблоны справочника **Печатные шаблоны**. В зависимости от раздела, из которого вызван мастер, производится фильтрация печатных шаблонов по их типу.

:::

Изначально в программе предоставлены стандартные шаблоны, которые недоступны для редактирования. Добавление пользовательских шаблонов в программу предусмотрено следующими способами: 

копирование стандартных или пользовательских шаблонов, с их последующим изменением;

импорт шаблона.

Рассмотрим данные способы подробнее.
